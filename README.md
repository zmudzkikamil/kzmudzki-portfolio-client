## Tech Stack

### Frontend (this repo)

| Layer           | Technology                      |
| --------------- | ------------------------------- |
| Framework       | React 18, React Router 7        |
| Language        | TypeScript 5 (strict)           |
| Styling         | Tailwind CSS 3                  |
| Data fetching   | TanStack React Query v5 + Axios |
| Build           | Vite 5                          |
| Tests           | Vitest + Testing Library        |
| Type generation | openapi-typescript              |

### Backend ([separate repo](https://github.com/zmudzkikamil/kzmudzki-portfolio-server))

| Layer           | Technology            |
| --------------- | --------------------- |
| Framework       | NestJS                |
| Database / BaaS | Supabase (PostgreSQL) |
| Language        | TypeScript            |

> **Note:** NestJS + Supabase is arguably overkill for a personal portfolio, and
> I know it. The stack was chosen intentionally for **learning purposes** -
> getting hands-on experience with NestJS architecture and Supabase outside of
> tutorial projects.
